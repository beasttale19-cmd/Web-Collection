# Web Collection

Web Collection is the copy of different yswss! of Hackclub.

## Yswss Copy : till now

- Macondo
- Flavortown
- Hack Club The Game

## Languages Used

- HTML
- CSS
- Some little JavaScript

## Demo

You can check the demo from: https://beasttale19-cmd.github.io/Web-Collection/main.html .

## How to use:

This isnt any kind of very useful project. I made this project to build my skill in Web Developement.
But to use: We can hover and click on any yswss copies to test it out!
